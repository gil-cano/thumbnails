thumbnails
==========

Create a collage from a bunch of images

Installation
------------

.. code:: bash

    $ git clone https://github.com/gil-cano/thumbnails.git
    $ cd thumbnails
    $ virtualenv-3.5 .
    $ source bin/activate
    (papiro)$ pip install -r requirements.txt

Use
---

.. code:: bash

    (papiro)$ python thumbnail.py
    (papiro)$ python collage.py
